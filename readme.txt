== Description ==
A Custom IFrame Widget,  You can use it in page, post or in widget  

Simply Copy and Paste this code in your page/post  

[ShaktiIFrame http://wordpress.opensourcedevelopers.net/ 650px 200px]  


[ShaktiIFrame url Width Height]  
Replace url with some other website url which you want to show on your website page/post.  
Change Width and Height of your IFrame.  
URL: http://wordpress.opensourcedevelopers.net/  
Width: 650px  
Height: 200px  


== Installation ==
= How to install Custom IFrame Widget (manual) =  
1. Download Custom IFrame Widget.  
2. Upload it into your plugins directory from Plugin Add Feature or FTP.  
3. It will create a directory /wp-content/plugins/custom-iframe-widget/  
4. Active Custom IFrame Widget.  

How to Use Custom Iframe Widget  
-------------------------------------
It can be set through widget and assign to any widget area.  

Configuration  
-------------
Title: Widget Title  
URL: http://wordpress.opensourcedevelopers.net/  
Width: 200  
Height: 200
